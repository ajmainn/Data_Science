Rain Prediction using australian weather dataset from Kaggle
This project is focused on building a predictive model to forecast whether it will rain the next day based on historical weather data collected from various locations in Australia. The dataset includes daily weather observations such as temperature, humidity, wind speed, pressure, rainfall etc.
The dataset contains daily weather observations from various Australian locations, including features such as temperature, rainfall, humidity, wind speed/direction, pressure, and cloud cover. The target variable is "RainTomorrow", indicating whether it will rain the next day.

  Algorithms used: Logistic Regression, Random Forest, XGBoost, etc.
  Evaluation metrics: Accuracy, Precision, Recall, F1-score, ROC-AUC

Language: R
Tools: R Studio
